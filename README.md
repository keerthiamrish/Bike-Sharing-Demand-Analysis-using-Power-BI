# Bike-Sharing-Demand-Analysis-using-Power-BI
## Project Overview

This project is a Power BI dashboard built to analyze bike sharing data and identify usage patterns, trends, and performance insights.

## Objectives
Understand bike usage trends
Analyze peak hours and demand
Identify factors affecting rentals
Improve decision-making using data insights

## Tools Used
Power BI
Data Visualization
Data Analysis

## Key Insights
Peak usage observed during specific hours
Demand varies based on time and conditions
Opportunities to optimize bike availability

## File Included
bike sharing.pbix – Power BI dashboard file
bike sharing .pdf - Power BI Report
